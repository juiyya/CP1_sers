# SERS – SOLUÇÕES EM ENERGIAS RENOVÁVEIS E SUSTENTÁVEIS

## A atividade propõe que a gente observe como escolhas de implementação impactam diretamente o consumo de energia de um sistema. 
- Programas feitos em Python simulando criações, buscas e remoções em: Linked List, Hash Map e List
- Profiling por meio de CodeCarbon e Tracemalloc (EmissionsTracker)
